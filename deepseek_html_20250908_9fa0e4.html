Vite Frontend Build Setup

Recommended build tool for your PVA Bazaar marketplace

Why Vite?

  • Extremely fast builds and hot reload
  • Simple configuration
  • Great for modern web apps
  • Optimized production builds
  • Excellent developer experience

Installation Command:

npm create vite@latest frontend -- --template vanilla

Alternative Options

  • Webpack - More powerful but complex configuration
  • Parcel - Zero configuration, good for simple projects
  • esbuild - Extremely fast but less feature-rich
  • Rollup - Great for libraries, more setup needed

Vite is the best choice for your project

Step-by-Step Vite Setup

1 Install Vite in your frontend directory
cd frontend npm create vite@latest . -- --template vanilla npm install
2 Update your root package.json
{ "scripts": { "start": "node server.js", "dev": "nodemon server.js", "build": "cd frontend && npm run build", "vercel-build": "npm run build" } }
3 Update vercel.json configuration
{ "version": 2, "builds": [ { "src": "package.json", "use": "@vercel/static-build", "config": { "distDir": "frontend/dist" } }, { "src": "server.js", "use": "@vercel/node" } ], "routes": [ { "src": "/api/(.*)", "dest": "/server.js" }, { "src": "/(.*)", "dest": "/$1" } ] }
4 Move your frontend files to the right structure
frontend/ ├── src/ │ ├── assets/ │ ├── components/ │ ├── pages/ │ ├── main.js │ └── style.css ├── index.html ├── package.json └── vite.config.js
